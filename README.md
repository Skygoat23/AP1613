# 📊 Nico Aronne AP 1613 — Equity Dashboard

Dashboard financiero personal para análisis de 13 empresas seleccionadas, construido con HTML/CSS/JS puro y Chart.js.

## 🔍 ¿Qué es esto?

Una herramienta de análisis financiero personal que permite explorar métricas clave de 13 empresas:

**CRWD · ANET · O · GLNG · SNDK · NBIS · HIMS · ONDS · COP · NEE · MP · CCJ · FISV**

## 📈 Funcionalidades

- **6 tabs por empresa** — Perfil, Negocio, Rentabilidad, Situación Financiera, Valuación, Ranking IA
- **Métricas clave** — Revenue, EBITDA, FCF, Márgenes, ROE, ROIC, P/E, EV/EBITDA y más
- **Gráficos históricos** — Evolución de todas las métricas por año
- **Múltiplos vs histórico** — Compara valuación actual contra promedio histórico
- **Análisis IA individual** — Claude analiza cada empresa con sus datos reales
- **Ranking IA comparativo** — Claude rankea las 13 empresas para inversión en dólares

## 🛠️ Tecnologías

- HTML / CSS / JavaScript puro (sin frameworks)
- [Chart.js 4.4.1](https://www.chartjs.org/) — licencia MIT
- [Claude API](https://www.anthropic.com/) — análisis con IA
